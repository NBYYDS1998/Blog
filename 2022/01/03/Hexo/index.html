安装Hexo教程

一、安装nodejs

nodejs 下载链接：https://nodejs.org/download/release/latest-v13.x/

二、安装git

git 下载链接：https://git-scm.com/downloads

三、终端运行命令安装Hexo

sudo npm install hexo-cli -g
